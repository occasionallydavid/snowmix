Introduction


This page list the Snowmix Guides. The Snowmix Guides are manuals each centered on a specific topic with detailed examples on how to use a specific group of features within Snowmix.

For now the following Snowmix Guides has been written:

  • Snowmix Audio Guide.
  • Snowmix AV Sync Guide.
  • Snowmix Basic Scripting Guide.
  • Snowmix Libraries Guide.
  • Snowmix Glshapes Guide. Snowmix OpenGL Support.
  • Snowmix Scenes Tutorial.
  • Snowmix Shapes Guide.
  • Snowmix Tcl Interpreter Guide.
  • Snowmix Text Guide.
  • Feeding Live Video to CDNs.

There will be written more manuals and the existing ones are expected to be updated and improved over time.

Video Tutorials


So far a number of video tutorials are available. You can run them your self executing the following commands:

snowmix ini/text_tutorials.ini

and then in second window

output2screen

and then in third window (although not strictly needed)

input2feed 4 4
